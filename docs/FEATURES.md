# AbyssOS Features

## Current Features (v1.1)

### File Management & Storage
- **500GB Per-User Storage**: Each AbyssID account receives 500GB of server storage
- **Drag-and-Drop Upload**: Files can be dragged from desktop/device directly to AbyssOS
- **Auto-Minting**: Uploaded files are automatically minted as DRC-369 NFTs on-chain
- **File Explorer**: View all uploaded files and NFTs with metadata
- **Remote Access**: Files accessible from anywhere when logged in with AbyssID

### NFT System
- **DRC-369 Standard**: Native NFT standard for all on-chain assets
- **Cross-Chain Swapping**: Swap NFTs from Ethereum, Solana, and Polygon to DRC-369
- **Wallet Signature Validation**: Secure ownership verification via wallet signatures
- **Metadata Preservation**: All NFT metadata and ownership information preserved

### Wallet & Payments
- **Abyss Wallet**: Integrated wallet with CGT balance tracking
- **5000 CGT Sign-Up Bonus**: New users receive 5000 CGT on registration (minted on-chain)
- **Send Restrictions**: Users must mint or swap an NFT before sending CGT
- **Transaction History**: Complete transaction tracking and history

### Media Playback
- **NEON Player**: Full-featured media player supporting:
  - All video formats (MP4, WebM, OGG, MOV, AVI, MKV, etc.)
  - All audio formats (MP3, WAV, OGG, FLAC, etc.)
  - Fractal-1 audio codec support
  - Background playback mode
- **Video Playback**: Full video player with controls
- **Audio Visualization**: NEON-reactive visualizations based on audio beatmaps

### Document Editing
- **Document Editor App**: Full document editing suite
- **PDF Viewing**: View PDF documents in-app
- **Text Editing**: Edit TXT, MD, JSON, XML, HTML, CSS, JS, TS files
- **Image Viewing**: View JPG, PNG, GIF, WebP, SVG images
- **Save Functionality**: Save edited documents back to storage

### User Interface
- **Start Menu**: Glowing orb start button with jewel-like texture
- **System Menu**: Categorical tabbed menu system for all applications
- **Window Management**: Minimize, maximize, resize, and drag windows
- **Status Bar**: Date/time, chain status, music player widget, avatar menu

### Blockchain Integration
- **RPC Connection**: Real-time blockchain connection status
- **Block Explorer**: View transactions and blocks
- **Chain Operations**: Direct chain interaction tools
- **Mining Support**: Integrated mining application

### Development Tools
- **Abyss Runtime**: WASM module execution environment
- **Abyss Shell**: Command-line interface
- **System Monitor**: System resource monitoring
- **Grid Monitor**: P2P network monitoring

### Docker Support
- **Development Containers**: Docker Compose setup for local development
- **User Workspaces**: Isolated development environments
- **Service Orchestration**: Chain, AbyssID Service, and Portal containers
