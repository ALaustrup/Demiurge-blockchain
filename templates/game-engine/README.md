# Demiurge Game Engine Template

Lightweight game client template with Demiurge Blockchain integration.

## Features

- ✅ C++ with Raylib
- ✅ UrgeID login
- ✅ NFT fetching
- ✅ CGT microtransactions
- ✅ 3D scene integration

## Quick Start

1. **Install dependencies:**
   ```bash
   # Install Raylib
   # On Ubuntu/Debian: sudo apt install libraylib-dev
   # On macOS: brew install raylib
   # On Windows: Download from raylib.com
   ```

2. **Build:**
   ```bash
   mkdir build && cd build
   cmake ..
   make
   ```

3. **Run:**
   ```bash
   ./demiurge-game
   ```

## Customization

This is a minimal template. Expand it to:
- Load and render NFTs as 3D objects
- Implement CGT transactions
- Add multiplayer features
- Integrate with Recursion Engine (when available)
